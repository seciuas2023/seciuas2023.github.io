WELCOME TO SecIUAS 2023!

The SecIUAS 2023 workshop is scheduled on November 7, 2023 and will be held at Nagoya, Japan.

SecIUAS 2023 will be co-located with the 13th International Conference on the Internet of Things (IoT 2023).

We solicit high-quality position papers and research papers that address opportunities and challenges in the areas of hardware security, intelligent system security, application security and network security. We welcome contributions to all relevant research on attacks on IUAS, hardware protections, secure software modelling and verification, security and safety analysis, network security and so on.

The topics of interest include, but are by no means limited to:

  • Attacks on IUAS including spoofing, eavesdropping, side-channel attack and so on
  • Adversarial machine learning on IUAS
  • Hardware’s vulnerabilities detection for IUAS
  • Security protection for IUAS based on TPM (Trust Platform Model)
  • Modelling, simulation and tools for secure and safe design of IUAS
  • Software’s vulnerabilities detection for IUAS
  • Automatic security and safety verification for IUAS software
  • Efficient authentication and key management between IUASs
  • End-to-End secure communication between IUASs
